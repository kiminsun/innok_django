# INNOK 홈페이지 제작 프롬프트 (요약 버전)

## 프로젝트 개요
**목적**: innok.kr 의료제품 GMP 교육 및 컨설팅 회사 공식 홈페이지 제작

---

## 페이지 구성
1. **메인 (INNOK)**: 회사 홍보 이미지(2개) 배경 + 상징적 멘트 오버레이
2. **사업소개**: 5가지 사업(특화/일반/교육/규제전문에이전시/M&A) - 갤러리 형식, 클릭 시 상세보기 팝업
3. **회사소개**: 회사 소개 멘트 + 대표 이미지
4. **고객센터**: 연락처 정보, SNS 링크, 지도
5. **공지사항**: 게시판(번호/제목/글쓴이/조회/등록일자)
6. **견적문의**: Q&A 게시판(번호/제목/글쓴이/비밀번호/조회/등록일자)
7. **자주하는질문**: Q&A 게시판(번호/제목/글쓴이/비밀번호/조회/등록일자/답변일자)
8. **PR CENTER**: 동영상 링크, 사진 갤러리
9. **푸터**: 정부기관 링크, 블로그/인스타, 오시는 길

---

## 디자인
- 모던하고 깔끔한 디자인 (최신 트렌드)
- 반응형 (모바일/태블릿/PC)
- 부드러운 스크롤 애니메이션
- 자연스러운 색조합, 그라데이션, 그림자 효과
- 로딩 애니메이션
- Hover 효과

---

## 주요 기능
- 원페이지 스크롤
- 고정 네비게이션 바 (스크롤 시 스타일 변경)
- 관리자 모드 (`/innokadm/`)

---

## 관리자 기능
1. **데이터 관리 (JSON)**: 사업소개, 회사소개, 고객센터 - 추가/수정/삭제, JSON 다운로드
2. **게시판 관리 (MySQL)**: 공지사항, 견적문의, 자주하는질문 - CRUD 기능

---

## 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5
- **Backend**: Django Framework
- **Database**: MySQL 8.0
- **배포**: GitHub Actions → GitHub Pages
- **추가 라이브러리**: AOS/GSAP (애니메이션), Swiper.js (슬라이더), Lightbox (갤러리)

---

## 데이터 구조
- **JSON**: 사업소개 프로젝트, 회사소개, 고객센터 정보
- **MySQL**: 공지사항, 견적문의, FAQ 테이블

---

## 추가 고려사항
- **보안**: SQL Injection 방지, XSS 방지, CSRF 토큰, 비밀번호 암호화
- **성능**: 이미지 최적화, 파일 압축, CDN, 캐싱
- **SEO**: 메타 태그, Open Graph, 구조화된 데이터, 사이트맵
- **접근성**: ARIA 레이블, 키보드 네비게이션, 스크린 리더 호환
